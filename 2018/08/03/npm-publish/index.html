省去千言万语，你自己点进去看吧

先看看怎么unpublish

It is generally considered bad behavior to remove versions of a library that others are depending on!

我们使用npm，都是云上的东西，如果你写的玩意正好被他人用着，哪天种子下不到了，的确会造成很多人的麻烦。
如果真的迫不得已，还是使用deprecate，了解更多卸载政策？。
